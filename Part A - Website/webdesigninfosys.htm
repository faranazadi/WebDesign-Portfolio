Web Design

This unit will introduce me to a range of topics in the area of web design and development. In the first term I will start by learning the basic building blocks of the web; HTML and CSS. By the end of this section I will have the skills to develop and style a simple website. In the second half of term 1 I will learn about HTML5 video and canvas, which are implemented using Javascript. In the second term I will learn how to add dynamic content to a website using MySQL and PHP. I will also look at content management systems (CMS).


Information Systems

An information system (IS) is any organized system for the collection, organization, storage and communication of information. More specifically, it is the study of complementary networks that people and organizations use to collect, filter, process, create and distribute data. "An information system (IS) is a group of components that interact to produce information"
